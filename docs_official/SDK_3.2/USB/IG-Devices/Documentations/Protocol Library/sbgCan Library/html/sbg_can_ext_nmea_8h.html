#define SBG_CAN_NMEA_OPT_HDT_AFTER_RMC   (0x0002)

If set the HDT frame will be time stamped with the PREVIOUS RMC time data.

#define SBG_CAN_NMEA_OPT_HDT_BEFORE_RMC   (0x0000)

If set the HDT frame will be time stamped with the NEXT RMC time data.

SbgErrorCode sbgCanExtNmeaSetYawOffset ( SbgCanDeviceHandle  handle,
float  offset 
)

Configures the offset between the IG-500E orientation and the GPS true Heading data.
IG-500E heading is then GPS True Heading - offset; This offset actually is stored as a rotation matrix.
See sbgCanExtNmeaSetMatrixOffset or sbgCanExtNmeagetMatrixOffset for more information

Parameters
[in]handleA valid sbgCom library handle.
[in]permanentIf TRUE, stores this settings in flash memory.
[in]offsetoffset between the GPS true heading and IG-500E heading
Returns
SBG_NO_ERROR if no error.
