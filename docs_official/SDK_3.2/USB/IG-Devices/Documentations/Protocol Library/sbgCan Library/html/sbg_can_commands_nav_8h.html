Enumeration Type Documentation

enum _SbgCanAidingPosSrc

Define the position aiding source to use in the Kalman Naviagation filter.

Enumerator
SBG_CAN_POS_SRC_GPS 

We are using the GPS position as the aiding source for the navigation filter

SBG_CAN_POS_SRC_GPS_AND_BARO 

We are using the GPS position and the barometric altitude as the aiding source for the navigation filter

SBG_CAN_POS_SRC_USER 

We are using the user provided source for position as the aiding source for the navigation filter

enum _SbgCanAidingVelSrc

Enumerator
