Enumeration Type Documentation

enum _SbgCanHeadingSource

List of available sources for heading corrections.

Enumerator
SBG_CAN_HEADING_SOURCE_NONE 

The heading is calculated using gyroscopes only.

SBG_CAN_HEADING_SOURCE_MAGNETOMETERS 

The heading is based on magnetometers information and gyroscopes.

SBG_CAN_HEADING_SOURCE_GPS_COURSE 

GPS course is used

SBG_CAN_HEADING_GPS_ACCELERATIONS 

GPS + accelerometer heading source

SBG_CAN_HEADING_SOURCE_USER 

Heading is fed by user via the main protocol

SBG_CAN_HEADING_SOURCE_REMOTE_MAG 

Remote magnetometers used as a heading source

SBG_CAN_HEADING_SOURCE_REMOTE_TRUE_HEADING 

Remote true heading sensor used (dual antenna)
