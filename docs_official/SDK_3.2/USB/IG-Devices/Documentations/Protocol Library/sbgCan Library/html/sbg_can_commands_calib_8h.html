enum _SbgCanCalibMagsAction

For the magnetometers calibration command, defines which action to execute.

Enumerator
SBG_CAN_CALIB_MAGS_LOAD_DEFAULT 

Load into volatile memory the default magnetometers calibration data.

SBG_CAN_CALIB_MAGS_SAVE 

Save the current magnetometers calibration data into FLASH memory.

SBG_CAN_CALIB_MAGS_WRONG_PARAMETER 

Action returned by the device when the input parameter is invalid.
